# tools-animate-asset-toolkit

**Asset Wizard for Adobe Animate** - React + Node.js application for generating text for banners and SVG/PNG graphics with support for Figma API, export to Animate and HTML conversion.

## Technology stack

- **Frontend:** React + Vite + TailwindCSS
- **Backend:** Express + TypeScript + Prisma
- **API/Integracje:** Figma REST API
- **Dev Tools:** tsx, concurrently, yup, nodemon, eslint, prettier

## How to run locally

```bash
# 1. Installation of dependencies in main + frontend + backend
npm run install-all

# 2.  Running the backend + frontend server simultaneously from the main folder
npm run dev
```
